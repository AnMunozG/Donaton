# Donatón
Plataforma web de donaciones transparentes para conectar donantes, municipalidades y equipos de logística en la ayuda humanitaria.
## Requisitos
- Node.js 18 o superior
## Instalación
``` Terminal
npm install
```

## Desarrollo

``` Terminal
npm run dev
```

Abrir [http://localhost:5173](http://localhost:5173) en el navegador.

## Producción

``` Terminal
npm run build
npm run preview
```

## Pruebas Unitarias


``` Terminal
npm run test
```
